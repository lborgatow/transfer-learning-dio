# :brain: Projeto de Transfer Learning em Python :brain:

- Projeto realizando durante o Bootcamp "Geração Tech Unimed-BH - Ciência de Dados", na [DIO](https://www.dio.me/);
- Consiste em aplicar o método de Transfer Learning em uma rede de Deep Learning na linguagem [Python](https://docs.python.org/3/) no ambiente [COLAB](https://colab.research.google.com/);
- O dataset utilizado engloba duas classes: gatos e cachorros. Uma descrição da base de dados pode ser visualizada clicando [aqui](https://www.tensorflow.org/datasets/catalog/cats_vs_dogs);

## :blue_book: Bibliotecas utilizadas:

- [Tensorflow](https://www.tensorflow.org/?hl=pt-brcd);
- [Keras](https://keras.io/);
- [Matplotlib](https://matplotlib.org/);
- [Pillow](https://pillow.readthedocs.io/en/stable/reference/Image.html).
